新規のプロジェクトに入る時の心構え

全体がわかっていないとコードを書くことが絶対わからない。 必ず新規プロジェクトに入った際には必ず仕様やドキュメントをしっかりみること。

プログラムを書くまえに考えること

書きながらロジックを構成していくのは上級者。 そうなるべきだが、まずは紙でも良いのでロジックを考える。
