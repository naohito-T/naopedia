## JSON Schema種類

>JSON ハイパースキーマは、2021 年の時点で休止中 / 現在メンテナンスされていません。
>これにより、チームは JSON スキーマのコアと検証に費やすわずかな時間を集中できます。
>後日、JSON ハイパースキーマを再検討する可能性があります。

これによりJSONハイパースキーマは種類に該当しなくていい。
