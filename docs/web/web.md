## web画面遷移の仕組み

webブラウザでは`<a>`タグで示されたハイパーリンクをユーザがクリックすると画面がリンク先に繊維して、閲覧履歴にリンク先が追加される。ここでユーザが戻るボタンを押下すると、もとの画面に遷移して閲覧履歴から先のリンク先ページが削除される(pop)。
