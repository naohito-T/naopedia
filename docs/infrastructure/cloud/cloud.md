## PaaS(Platform as a Service)

IaaSではネットワーク・ハードウェア（マシン本体）・OSまでをセットにして提供するのに対し、**PaaSではそれらに加えてミドルウェアまでを提供します。**
Web開発ではサーバーソフトウェアやデータベースソフトウェアのことと同義。
