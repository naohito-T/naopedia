# Typings

- Typingsとは

**tsdと同じくTypeScriptの型定義ファイルを管理するツール**
Githubから型定義を取得してくるしかなかったtsdと違って、npmやbower、githubプロジェクト、ローカルファイルシステムやhttp経由で型定義ファイルを取ってこれるのが特徴で
>tsdと違ってGitHubのAPIを叩かない
