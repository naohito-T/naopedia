## useState

関数が状態を持てるようになる.

関数とは？もともとの定義って決まった処理を返す。じゃなかったっけ。
それに状態をもたせるようにした。
一番トップのコンポーネントでuseStateを宣言して子コンポーネントで共有する。
→用は一つの関数内で状態を管理する。
